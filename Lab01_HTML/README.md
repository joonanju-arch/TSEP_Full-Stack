# Lab 1 - HTML Basics

## Objective
Create your first HTML webpage.

## Tasks

1. Create index.html
2. Add your Name
3. Add your Roll Number
4. Add your Branch
5. Add College Name
6. Add one image
7. Add one hyperlink
8. Add an unordered list
9. Add a table with three subjects

1. Push code to GitHub.
2. Publish using GitHub Pages.
3. Submit Repository URL.
4. Submit Live Website URL.
